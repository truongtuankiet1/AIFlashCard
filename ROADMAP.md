# VocabCards - Development Roadmap

This document outlines the features and improvements planned for VocabCards.

## Current Status: MVP (Phase 1) ✅

### Completed Features
- ✅ User authentication (sign up, sign in, sign out)
- ✅ AI-powered flashcard generation using OpenAI
- ✅ Interactive flashcard UI with flip animation
- ✅ Spaced Repetition System (SM-2 algorithm)
- ✅ Deck management (create, view, delete)
- ✅ User progress tracking
- ✅ Dashboard with statistics
- ✅ Settings page for API key management
- ✅ CSV import/export
- ✅ Text-to-speech pronunciation

## Known Limitations

- Single API provider (OpenAI) - plan to support Claude, Gemini, etc.
- No offline functionality yet
- Limited to English and Vietnamese definitions
- Basic analytics only
- No video content support
